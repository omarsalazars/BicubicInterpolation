# Image Interpollation
Image interpollation using these methods:
* Nearest Neighbor
* Bilineal Interpolation
* Bicubic Interpolation
* Average

## Run
```
git clone git@github.com:omarsalazars/BicubicInterpolation.git
cd BicubicInterpolation
make
./interpolation
```
